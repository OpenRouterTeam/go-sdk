title: "Value2"

## Supported Types

### 

```go
value2 := operations.CreateValue2Str(string{/* values here */})
```

### 

```go
value2 := operations.CreateValue2Number(float64{/* values here */})
```

## Union Discrimination

Use the `Type` field to determine which variant is active, then access the corresponding field:

```go
switch value2.Type {
	case operations.Value2TypeStr:
		// value2.Str is populated
	case operations.Value2TypeNumber:
		// value2.Number is populated
}
```

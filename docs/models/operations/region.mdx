---
title: "Region"
---

Filter to models with endpoints in the given data region. Currently only "eu" is supported.

## Example Usage

```go
import (
	"github.com/OpenRouterTeam/go-sdk/models/operations"
)

value := operations.RegionEu
```

| Name       | Value      |
| ---------- | ---------- |
| `RegionEu` | eu         |

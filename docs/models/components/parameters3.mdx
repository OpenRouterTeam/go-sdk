## Supported Types

### 

```go
parameters3 := components.CreateParameters3Str(string{/* values here */})
```

### 

```go
parameters3 := components.CreateParameters3Number(float64{/* values here */})
```

### 

```go
parameters3 := components.CreateParameters3Boolean(bool{/* values here */})
```

### 

```go
parameters3 := components.CreateParameters3ArrayOfParameters1([]*components.Parameters1{/* values here */})
```

### 

```go
parameters3 := components.CreateParameters3MapOfParameters2(map[string]*components.Parameters2{/* values here */})
```

## Union Discrimination

Use the `Type` field to determine which variant is active, then access the corresponding field:

```go
switch parameters3.Type {
	case components.Parameters3TypeStr:
		// parameters3.Str is populated
	case components.Parameters3TypeNumber:
		// parameters3.Number is populated
	case components.Parameters3TypeBoolean:
		// parameters3.Boolean is populated
	case components.Parameters3TypeArrayOfParameters1:
		// parameters3.ArrayOfParameters1 is populated
	case components.Parameters3TypeMapOfParameters2:
		// parameters3.MapOfParameters2 is populated
}
```

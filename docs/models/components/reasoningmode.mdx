---
title: "ReasoningMode"
---

Selects the reasoning mode. `standard` is the default; `pro` engages deeper reasoning on models that support it, billed at standard token rates. Only supported by OpenAI GPT-5.6 and newer.

## Example Usage

```go
import (
	"github.com/OpenRouterTeam/go-sdk/models/components"
)

value := components.ReasoningModeStandard

// Open enum: custom values can be created with a direct type cast
custom := components.ReasoningMode("custom_value")
```


## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `ReasoningModeStandard` | standard                |
| `ReasoningModePro`      | pro                     |
